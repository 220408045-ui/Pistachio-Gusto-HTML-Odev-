# PISTACHIO GUSTO Web Project

This project is a fictional corporate website for the "PISTACHIO GUSTO" brand, which produces luxury pistachio-based products (Dubai Chocolate, pistachio cream and paste). The goal of the assignment is to demonstrate the use of structural and semantic elements using **pure HTML**, without the use of CSS or JavaScript.

## File Organization
The project consists of four main HTML files and a media folder, all linked together by relative paths:

* **index.html**: Home Page. `<header>`, `<main>`, `<article>`, `<img>` ve `<footer>` içerir.
* **about.html**: Brand Story. `<section>`, `<figure>`, `<figcaption>` ve zorunlu **5 anlamsal etiket** (`<time>`, `<mark>`, `<blockquote>`, `<details>`, `<summary>`) içerir.
* **services.html**: Product Types. `<table>`, `<thead>`, `<tbody>`, `<tfoot>` yapısı ve en az 4 sütun / 5 satır içerir.
* **contact.html**: Contact Form. Tam donanımlı form öğelerini (`<form>`, `<fieldset>`, `<legend>`, `<label>` + farklı `<input>` tipleri, `<textarea>`, `<select>`) içerir.
* **assets/**:Image, audio and video files (img src="" ile referans verilen) folder in which it is stored.

## Challenges I Faced

The biggest challenge of the project was trying to solve the need for a modern look with the structural power of HTML, without using any CSS, while maintaining consistent navigation (`<nav>`) and page structure throughout the site.
